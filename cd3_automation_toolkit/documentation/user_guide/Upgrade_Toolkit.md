# Steps to Upgrade Your Toolkit (For Existing Customers):

## Release v10
1. Build new image using the latest code from main branch.
1. Launch a new container using the same outdir.
1. Use new excel sheet templates to use OKE and SCH.
1. For existing services, user can continue using existing outdir.
